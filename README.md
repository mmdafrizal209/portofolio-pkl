# Personal Portfolio Website - Moch Afrizal Permana Harun

Situs portfolio personal interaktif yang dibangun untuk menampilkan profil, keahlian teknis, serta proyek pengembangan aplikasi web dan mobile yang telah saya kerjakan selama masa studi di SMKN 1 Cianjur.

## 🚀 Fitur Utama

- **Desain Modern & Aesthetic:** Menggunakan skema warna *dark mode* yang nyaman di mata dengan sentuhan aksen gradasi yang dinamis.
- **Responsif penuh:** Dioptimalkan untuk berbagai ukuran layar mulai dari *smartphone*, tablet, hingga perangkat desktop menggunakan Tailwind CSS.
- **Komponen Interaktif:** Dilengkapi dengan *progress bar* visual pada bagian keahlian teknis dan mikro-interaksi pada kartu project.

## 🛠️ Keahlian Teknis (Tech Stack)

- **Frontend & Web:** HTML5, CSS3, JavaScript, Tailwind CSS, Next.js
- **Mobile Development:** Flutter, Dart
- **Backend & Database:** PHP, Laravel, MySQL

## 📁 Ringkasan Project & Karya

### 1. Afrizal Cell (UMKM Project)
Aplikasi *Point of Sale* (POS) yang dirancang khusus untuk manajemen stok handphone, aksesoris, serta pelacakan status servis unit pada toko seluler lokal.
- **Teknologi:** Flutter, Dart, Laravel API

### 2. Animated Landing Page (Frontend Project)
Sebuah platform *landing page* modern yang dibangun di atas framework Next.js dengan fokus pada performa tinggi dan interaktivitas tingkat lanjut. Mengimplementasikan teknik rendering modern, optimasi komponen, serta integrasi pustaka animasi untuk menciptakan pengalaman pengguna yang dinamis dan estetik.
- **Teknologi:** Next.js, Tailwind CSS, JavaScript
